## Can I connect Confluence / Notion / Google Drive?

We're gradually adding connectors for popular third-party engineering tools. Updates will be announced as new integrations become available.
Currently, we support Confluence and GitHub connectors for ingesting documentation and repositories. All ingested content is linked to your Git organization, ensuring transparency and control.

## Does ctx| scan my laptop?

No. ctx| indexes **repositories and knowledge tools you register** (and data derived from them in
your deployment). It does not install a generic endpoint agent on developer
machines as part of the core stack described here.

## How do private repositories work?

You connect ctx| to your GitHub account and organization to authorize all, or specific repositories to be indexed and stored in your Git account. 

## Can one user belong to multiple organizations?

Yes. Users can be members of many orgs; the **active organization** is carried on
the session, and URLs use `/:orgSlug/`. MCP requires an explicit `orgSlug` query
parameter. See [Managing your organization](/docs/managing-organization).
