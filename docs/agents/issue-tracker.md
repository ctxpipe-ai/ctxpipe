# Issue tracker: Linear

Issues and PRDs for this repository live in Linear.

## Conventions

- Create and manage issues in the team/project configured for this repo in Linear.
- Treat Linear as the source of truth for status and prioritisation.
- Include Linear issue IDs in branch names, pull requests, and commit messages where relevant.

## When a skill says "publish to the issue tracker"

Create a Linear issue in the configured team/project.

## When a skill says "fetch the relevant ticket"

Fetch the issue from Linear using the provided issue key or URL.
